The Keyhole Laser Absorption Homepage

The Keyhole Absorption Dataset

A fragment of the laser energy absorption datasets is openly available at the NIST Public Data Repository. Full absorption datasets can be made available upon reasonable request to B.J.S. or R.J.

The Keyhole Segmentation Dataset

Dataset for keyhole semantic segmentation is available through this google drive link.

The model training repo

The code and model architectures used in the end-to-end approach and semantaic segmentation models is available in this github repo.
